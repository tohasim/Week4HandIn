Answers:
  • What is a promise? A promise is a way to control the flow in your program, by making your program work asynchronous, and then doing your code in correct order
  • What does the fetch method return? The fetch method returns a promise, and that promise resolves in a Response.
  • Why do we need to have two .then callbacks in our code? We need 2 callbacks to make sure the code is parsed correctly
  • What can and should you do in the first? Parse the response to JSON
  • What can and should you do in the last? Manipulate the data from the response (which is now in JSON)
